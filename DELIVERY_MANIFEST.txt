🚀 QUICK START
================================================================================

1. CREATE PROJECT
   npm create vite@latest sacco-web -- --template react
   cd sacco-web

2. INSTALL DEPENDENCIES
   npm install
   npm install zustand @tanstack/react-query react-hook-form zod axios recharts
   npm install -D typescript @types/react @types/react-dom

3. COPY FILES
   Copy all generated .jsx, .ts, .js files to src/
   Copy config files (vite.config.js, tsconfig.json, etc) to root

4. SET ENVIRONMENT
   echo 'VITE_API_URL=http://localhost:8000/api' > .env

5. RUN DEVELOPMENT SERVER
   npm run dev
   Visit: http://localhost:3000

Total Time: ~5 minutes

🔧 BACKEND INTEGRATION
================================================================================

Connect to Laravel API by:

1. Update .env with API URL
   VITE_API_URL=http://localhost:8000/api

2. Laravel should provide:
   ✅ Auth endpoints (login, register)
   ✅ Member endpoints (profile, savings)
   ✅ Loan endpoints (apply, list, details)
   ✅ Payment endpoints (M-Pesa integration)
   ✅ Downloads endpoints (resources)
   ✅ Reports endpoints (statements)

3. Configure CORS in Laravel
   CORS_ALLOWED_ORIGINS=http://localhost:3000,http://localhost:5173

4. Test API integration
   npm run dev
   → Try login
   → Check Network tab for API calls
   → Verify token in headers
